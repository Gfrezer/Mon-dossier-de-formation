Les Cichlidés nains américains.

La famille des Cichlidés présente une étonnante diversité de morphologies et couleurs selon les espèces. C’est le cas en Amérique du Sud où se côtoient les rios, le gros oscar ou encore les superbes petits apistos et acaras. Partons à la découverte de ces poissons calmes à mettre dans presque tous les aquariums !.
